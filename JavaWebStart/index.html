Webstart

For this to work, you must satisfy the following conditions:

  • You must install a recent version of java (1.4.2), which you can get here. Download it and install it.
  • You will have to get some annotation files, as we do not deliver these with the webstart package. You can get them here. Get the annotation file(s) for the microarray platform(s) you are using. Unzip the files and put them somewhere you can find them. Do the same for the GO XML file, which is found here (get the file named go_[date]-termdb.xml.gz). In the first step of running ermineJ, you will be asked to locate these files for the software.
Click here to launch ermineJ

Note that there can be some problems with Webstart message windows coming up at the same time at ermineJ is trying to display dialog boxes. If you find yourself confronted with a dialog box nagging you about destop integration, but you can't click on it, try clicking on the ermineJ dialog box underneath and then going back to the Webstart.
