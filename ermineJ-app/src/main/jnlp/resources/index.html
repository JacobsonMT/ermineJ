Webstart

For this to work, you must satisfy the following conditions:

  • You must install a recent version of java (1.6 or later), which you can get here. Download it and install it.
  • You will have to obtain a GO XML file and a gene annotation file. You can get them here. Get the annotation file(s) for the genes/probes you are using. Do the same for the GO XML file, which is found here. In the first step of running ermineJ, you will be asked to locate these files for the software.
