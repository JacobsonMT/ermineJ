Webstart

  • You must install a recent version of java (1.8 or later), which you can get here. Download it and install it.
  • You must be able to get a GO XML file (which ErmineJ will try to install for you) and gene annotations for your study. You can get the latter from here. Get the annotation file(s) for the genes/probes you are using. If ErmineJ cannot get the GO file, download it from the GO archives.
