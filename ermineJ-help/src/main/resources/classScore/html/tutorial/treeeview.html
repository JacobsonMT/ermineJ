The ErmineJ Tree view

Important to know about

  • Double-click on a node to view the details. Single-click on a node to expand its immediae children.
  • Right-click (MacOS:option-click) on a node to get more tools.
  • If you have more than one result set loaded, you can choose the one to show in the tree using the "Results" menu.
  • ErmineJ has the concept of 'redundancy' between groups. If two (or more) GO terms have exactly the same members, only one will be used for analysis. The redundant terms are not shown in the tree by default, but can be revealed using the pop-up menu.
  • Similarly, terms which have no genes in your annotations are hidden by default but can be shown using the pop-up menu.
