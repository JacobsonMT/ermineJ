Input files: Gene Ontology XML

ErmineJ currently uses the Gene Ontology RDF XML* file to obtain a complete set of data about GO terms and their definitions. This file is created by the Gene Ontology team, so the file names mentioned here are subject to change without notice. We distribute a recent version of the required file with the ermineJ distribution (unless you are using the webstart version).

The most recent version of the GO file should be available from the GO web site here. This file can be placed in your ermineJ.data directory for convenience. There is no need to unzip the file.

If the link above is broken, go to http://www.godatabase.org/dev/database/ and look for the download link -- and let us know.

Older versions of the GO files should be available as well. Please report problems with broken links on the GO web site to the GO web site maintainers or the go-database mailing list.

Note that the  annotation file you use is not assured to be 'in synch' with the GO XML file you download. This matters because the GO terms change, and it is possible for an annotation file to have terms which are newer than the GO XML file you use.

If you are using the annotation files we provide, you are advised use the GO XML file that was used during the latest annotation build. A link to that file is maintained at http://bioinformatics.ubc.ca/microannots/ .

If you are using an annotation file provided by a third party, or which you made yourself, we advise that you try to use the same version of GO that was used to created the annotations.

*If you don't know what RDF or XML mean, don't worry, you don't need to know.
