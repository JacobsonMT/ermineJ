Input files: Gene Ontology

ErmineJ currently uses the GO XML file to obtain a complete set of data about GO terms and their definitions. We distribute a recent version of this file with the software. If you want to upgrade to a newer version, go to http://www.godatabase.org/dev/database/archive/latest/ and select the latest "termdb.rdf-xml" file (not the very large assocdb file). This file should be unzipped in your ermineJ.data/genesetdata directory.

Note that our annotation files are not automatically assured to be 'in synch' with the file you download. Therefore you are advised use the GO XML file that was used during the latest annotation build. A link to that file is maintained at http://microarray.cpmc.columbia.edu/annots/ .
