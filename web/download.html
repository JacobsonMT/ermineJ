Download and Installation


Requirements

ermineJ requires a Java 2 Runtime Environment (JRE) 1.4 or newer to run. You can download the platform-specific packages from SUN's website (http://java.sun.com/j2se/). Users of MS Windows NT/2000/XP can alternatively download the out-of-the-box package below (which contains a bundled JRE 1.4.2_05).
