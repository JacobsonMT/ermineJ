Gene Ontology analysis for microarray data

Overview

ErmineJ performs analyses of gene sets in expression microarray data. A typical goal is to determine whether particular biological pathways are "doing something interesting" in the data. The software is designed to be used by biologists with little or no informatics background. A command-line interface is available for users who wish to script the use of ermineJ.

Key features include:

  • Implementation of several methods for gene set analysis:
    • Over-representation analysis
    • A resampling-based method that uses gene scores
    • A resampling-based method that uses correlation between gene expression profiles .
  • Gene sets receive statistical scores (p-values), and multiple test correction is supported.
  • Support of the Gene Ontology terminology
  • User files use simple text formats.
  • Users can modify gene sets or create new ones.
  • The results can be visualized within the software.
  • It is simple to compare multiple analyses of the same data set with different settings.
  • Hyperlinks are provided to external sites to allow more efficient browsing of the results.

Run it now

You can run ermineJ from any operating system that supports Java (in theory) without running the installer by using java webstart. If you have Webstart installed, you run ermineJ right now.

This a good way to get the latest version, as we will keep the link updated with the most recent development build. On the other hand, this version should be considered less stable than the other releases.

Documentation

The software comes with on-line help. The manual is also available online (webstart users will have to use this option for now). If you are having problems with the software please read the FAQ. If your question isn't answered in the FAQ or in the manual, feel free to contact us

Download and installation

Requirements

ermineJ requires a Java 2 Runtime Environment (JRE) 1.4 or newer to run. You can download the platform-specific packages from SUN's website (http://java.sun.com/j2se/ ). Users of MS Windows NT/2000/XP can alternatively download the Windows installer with bundled JRE package below. On machines with less than 256 Mb of RAM ermineJ may suffer from poor performance.

Software Files

Download the appropriate file for your platform:

Description File Size
Windows installer with bundled JRE ermineJ-setup-jre.exe ~30 MB
Windows installer without JRE ermineJ-setup.exe ~7 MB
Generic bundle ermineJ-2.1-bindist.tar.gz ~7 MB
Source code ermineJ-2.1-src.tar.gz < 1 MB

Annotation files

To use the software, you have to have microarray annotation files and a recent Gene Ontology XML file. ErmineJ is distributed with sample files so most users can get started faster, but you will either need or want to get different files.

Go here to get annotation files.  Copy them somewhere you can find them. It is not necessary to unzip them.

Go here to get the XML file (something like go_200503-termdb.rdf-xml.gz, found here: http://www.godatabase.org/dev/database/archive/latest/; the date portion of the name may have changed). It is not necessary to unzip the file.

Note that you can define your own gene annotation files or use some third-party file formats. You can also define your own gene sets using ermineJ or import them into ermineJ.

Installation

On MS Windows NT/2000/XP, simply run the installer.

For other platforms, you should be able to use the "Generic bundle". There is a README file included that gives some instructions.
