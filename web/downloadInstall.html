Download and installation

Requirements

ermineJ requires Java 1.6 or newer. You can download Java for free from Oracle. On machines with less than 512 Mb of RAM ermineJ may suffer from poor performance.

If you use ermineJ, we strongly recommend that you subscribe to the ermineJ users mailing list . This gives you the fastest access to information about updates as well as providing us with a type of "registration" to help us measure the number of users of ermineJ.

Run ermineJ now with Webstart

You can run ermineJ from any operating system that supports Java without running the installer by using java webstart.

If you have Webstart installed, you run ermineJ right now.

Software Files

The standard bundle includes a GO XML file and two sample microarrray annotation files for mouse and human Affymetrix microarrays. See the "Annotation files" for information on how to get other annotation files.

Annotation files

To use the software, you have to have gene annotation files and a recent Gene Ontology XML file.

Go here to get gene annotation files.  Copy them somewhere you can find them. It is not necessary to unzip them. You may also use Affymetrix or Agilent supplied files.

As of version 3.0, ErmineJ will install the most recent GO file for you. If this fails, or if you want to use a different file, look here to get the XML file ( go_daily-termdb.rdf-xml.gz ). It is not necessary to unzip the file.

Note that you can also define your own gene sets using ermineJ or import them into ermineJ. For example, in this manner ErmineJ supports the use of MolSigDb "GMT" files (gene symbol version).

Installation

Perhaps the easiest way to run ermineJ is by using webstart, but it can be installed manually.

On MS Windows Vista/7/XP, simply run the installer and follow the instructions.

On Mac OSX, unpack the .zip file and drag the resulting "ermineJ" application bundle to your Application directory.

For other platforms (Linux), if you don't want to use Webstart, you should be able to use the "Generic bundle". There is a README file included that gives some instructions. The Generic bundle is also appropriate for command-line users of ErmineJ.
